Este repositorio contiene la primera fase del proyecto integrador para la asignatura FrontEnd 1.

Instrucciones de Uso
Para probar el sistema de inicio de sesión sigue estos pasos:
1. Descargue el repositorio o clone el proyecto.
2. Abra el archivo `index.html` en cualquier navegador.
3. Pulse el botón **"Ingresar al Sistema"**.
4. Importante: Abra la consola del navegador (F12) para visualizar la impresión de datos solicitada.
